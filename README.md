> Bash script that give you the daily nasa astronomy picture.

## Prerequisites

- wget

## Install

Edit settings on top of the script : DIR & CMD

Then run :

```sh
make install
```
Note for Mac users : run "make PREFIX=/usr/local install"

## Usage

```sh
apod -h
```
